Interaktive sett som ligner det reelle Inspera-formatet — åpne spørsmål med modellsvar og selvvurdering, flervalg med umiddelbar tilbakemelding. Tracker nederst i hjørnet teller poeng og karakter.

EKSAMEN 01
Bredt sett — alle ti kapitler
28 oppgaver · 100 poeng · 4 timer · 22 åpne + 6 flervalg
Hovedvekt på OWASP-anvendelse, kryptografi, autorisasjon og GDPR. Inkluderer et Flask-kode-quiz og en STRIDE-analyse av en chat-app.
→ EKSAMEN 02
Multi-level, JWT-angrep og PoisonGPT
27 oppgaver · 100 poeng · 4 timer · 21 åpne + 6 flervalg
Tester Kerckhoffs, Bell-LaPadula/Biba, JWT alg=none + key confusion, zero trust i mikrotjenester, og PoisonGPT-angrepets fem trinn. Kode-quiz er en Django views.py med IDOR, path traversal og SQLi via raw().
→ EKSAMEN 03
Gjesteforelesninger — misuse cases, SLSA, agentic AI, Privacy by Design
28 oppgaver · 100 poeng · 4 timer · 21 åpne + 7 flervalg
Tyngdepunkt på underrepresenterte temaer: misuse cases, attack trees med AND/OR, FIDO2/WebAuthn phishing-resistens, agentic AI og Excessive Agency, SLSA + sigstore, Cavoukians 7 PbD-prinsipper, tredjelandsoverføring (Cloud Act / DPF), semantisk SAST (CodeQL) og adversarial ML. Kode-quiz er en Node.js/Express-endepunkt med path traversal, usikker YAML-deserialisering og command injection.
→ EKSAMEN 04
Underrepresenterte vinklinger — XXE, padding oracle, OAuth state, taint analysis
25 oppgaver · 100 poeng · 4 timer · 18 åpne + 7 flervalg
Fokus på temaer som har stått lite igjen i tidligere sett: psychological acceptability + complete mediation, XXE og file upload, prototype pollution, Diffie–Hellman/MitM, padding oracle og KDF, OAuth state + redirect_uri, capability vs ACL, JWT algorithm confusion, DFD med trust boundaries, McGraws touchpoints, sikker code review, slopsquatting, indirekte prompt injection i RAG, lockfiles og reproducible builds, mTLS i service mesh, DPO + DSAR, biometri-grunnlag (Art. 6/9), og taint analysis (source/sink/sanitizer). Kode-quiz er Python Flask med pickle-deserialisering, SSRF og XXE.

Hvordan settet fungerer: klikk for å åpne ett sett. Hvert åpent spørsmål viser modellsvar når du folder ut "Vis modellsvar" — etter at du har sammenlignet med ditt eget svar, velger du selv hvor mye du fikk rett (Full pott / Nesten alt / Halvveis / Lite / Ingenting). Flervalg gir umiddelbar grønn/rød tilbakemelding ved klikk. Poeng og karakter (A–F) oppdateres fortløpende i tracker-en nederst til venstre.
